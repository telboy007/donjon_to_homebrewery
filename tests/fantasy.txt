## Description
The dungeon has the following features, these may include skill checks to perform certain actions.
{{descriptive
#### General Features
| Type | Detail |
|:--|:--|
| Floors | Uneven Flagstone (difficult to run) (dense rubble (move at half speed)) |
| Walls | Superior Masonry |
| Temperature | Cool |
| Lighting | Shadowy (phosphorescent fungus or candles every 20 ft.) |
| Special | None |
}}
Some of the corridors marked on the map have special features detailed below.
